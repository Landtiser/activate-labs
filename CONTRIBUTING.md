# Contributing

## Ways to Contribute

**Bug reports:** [Open an issue](../../issues) with details about what broke.

**Improvements:** Submit a PR for clearer instructions, better troubleshooting, or UX fixes.

**New labs:** Use `templates/lab-template.html`. Requirements:
- Solves a real workplace problem
- 20-60 minutes to complete
- No coding/server setup required
- Browser-based
- White-label ready

**Translations:** Open an issue to coordinate.

## Submission Process

1. Fork the repo
2. Create a feature branch
3. Make changes
4. Test with someone unfamiliar with the topic
5. Submit PR

## Lab Quality Checklist

- [ ] Clear objective
- [ ] Step-by-step instructions with screenshots
- [ ] Time estimate and difficulty level
- [ ] Troubleshooting section
- [ ] White-label customization points marked
- [ ] Mobile-responsive and accessible
- [ ] Print-friendly

Be kind. Be constructive.
